## How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/loopsaloy/sudoku-web-app.git
```
2. Navigate into Project Directory:
```bash
cd sudoku-web-app
```
3. Run the application:
```bash
python app.py
```
4. open your browser and go to:
```bash
http://127.0.0.1:5000
```
